# USB Hub Switch

The USB hub switch allows access control through disabling USB peripherals, like a computer's keyboard and mouse. Each of the 4 downstream USB ports can be individually switched, providing more levels of control over what computer peripherals different users have access to. For instance, every user may have access to the computer's keyboard and mouse, but only staff have the ability to print to the attached (USB) printer. 

A 5th auxiliary USB port is provided that is connected when the switch is in the off state. This can be used to connect a simple keyboard emulator to execute some cleanup PowerShell functions when the user is done with the machine.
