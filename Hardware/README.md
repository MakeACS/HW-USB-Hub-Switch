# Hardware

This section of the repository contains information relating to component-level electrical hardware.

In-progress work can be found in the EasyEDA Pro .eprj2 file under the "Source" folder, Falstad simulation .txt files can be found in the "Simulations" folder, and outjob files (PDF schematic, Gerber, BOM, etc.) for the most recent version can be found in the "Outjob" folder.
